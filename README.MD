# Diabetes Mellitus Type 2 - Mobile Application

Mobile Application design and development aiming at empowering adults with T2DM in self-disease management. The mobile app includes a digital diary facilitating entering and storing important information related to the level of achieving glycemic control (e.g glucose), lifestyle habits (e.g. physical activity, diet), blood pressure and medication for the management of DM and comorbidities (e.g insulin, antihypertensive drugs).
